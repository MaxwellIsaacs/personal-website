Overview

A fast Sudoku game built in Rust and compiled to WebAssembly for web deployment. The engine uses backtracking algorithms with constraint propagation to generate valid puzzles and provides real-time validation with an immediate-mode GUI.

Built with egui for cross-platform compatibility - the same Rust codebase runs natively on desktop and in browsers via WASM. Features multiple difficulty levels, keyboard/mouse controls, and optimized rendering for smooth 60fps gameplay.
